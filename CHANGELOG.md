## [1.0.5] - 14.03.2026

### Добавлено
- В плеер добавлена кнопка лайка в виде сердечка с двумя состояниями для быстрого добавления текущего трека в избранное.
- На detail-странице трека добавлена отдельная кнопка лайка для быстрого добавления трека в избранное.

### Исправлено
- Ужесточена очистка audio-потока при удалении и переключении треков: плеер больше не должен провоцировать каскадное создание `WebMediaPlayer` и соответствующий шторм ошибок в консоли браузера.
- Страница `Треки` теперь корректно снимает свои глобальные обработчики при уходе с маршрута, поэтому повторные заходы больше не накапливают `soundhub:library-updated` listeners и не запускают каскадные повторные перерисовки после удаления трека.
- Гидрация длительностей на странице треков ограничена по параллелизму и игнорирует устаревшие рендеры, чтобы удаление активного трека не порождало сотни фоновых `Audio`/`WebMediaPlayer`.
- Если сервер возвращает `401` из-за истекшей сессии, интерфейс теперь сразу переводит пользователя на экран логина без необходимости вручную делать `F5`.
- После logout или истечения сессии SPA теперь принудительно сбрасывает защищенный маршрут и history на `/`, поэтому `F5` больше не оставляет пользователя на закрытой вкладке и поведение соответствует zero-trust.
- После logout/протухания сессии локально сохраненное состояние плеера теперь очищается сразу, чтобы zero-trust сценарий не оставлял хвосты предыдущего сеанса.
- Во время активного прослушивания музыки сессия теперь продлевается автоматически, даже если пользователь не двигает мышью и не взаимодействует с интерфейсом.
- Во время batch-загрузки музыки сессия теперь тоже удерживается активной, поэтому пользователя больше не должно выбрасывать посреди долгого upload.
- Блок текущего трека в нижнем плеере получил явный hover/focus affordance и клавиатурную навигацию, поэтому теперь заметно, что по нему можно открыть detail-страницу трека.
- Для загрузки музыки добавлен режим `Заменять дубликаты`: система может либо пропускать дубликаты, либо удалять старую копию и ставить на ее место новый файл.
- Добавлена серверная проверка целостности аудиофайлов при upload и отдельная ручная проверка библиотеки на битые/нечитаемые треки через integrity report.

### Инфраструктура
- Версия приложения обновлена до `1.0.5`.
