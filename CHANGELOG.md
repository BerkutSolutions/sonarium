## [1.0.1] - 2026-03-12

### Добавлено
- Полноценные detail-страницы для альбомов, исполнителей, треков и плейлистов с собственными URL.
- Встроенное редактирование сущностей без старых модальных сценариев.
- Загрузка обложки исполнителя.
- Поле `description` для пользовательских плейлистов.
- Отдельная страница жанров и разбиение треков по жанрам.
- Страница профиля пользователя и просмотр профилей.
- Базовая система шаринга и совместного доступа к сущностям.
- Zero-trust авторизация с первым пользователем-администратором.
- Админская страница пользователей и управление регистрацией.
- Проверка обновлений в настройках для администратора.
- Поиск дублей по названию на странице треков.
- Объединение исполнителей и альбомов через интерфейс.
- Глобальный индикатор загрузки файлов и папок с возможностью отмены.
- Блок статистики по занимаемому музыкальной библиотекой месту в настройках.
- Отображение версии приложения в боковом меню.
- Настройка автоматической проверки обновлений для администратора.

### Изменено
- Полностью переработан интерфейс плеера: верхний слой прогресса, очередь, hover-состояния, drag-and-drop перестановка треков.
- Основные страницы приложения приведены к единому визуальному стилю.
- Sidebar и branding обновлены под `Berkut Solutions / Sonarium`.
- Экраны входа и регистрации переведены на отдельный branded auth-flow.
- Библиотека, настройки, профиль и пользователи получили новый page-shell и более цельный визуальный стиль.
- Загрузка файлов ускорена за счёт параллельной отправки и облегчённой серверной постобработки.
- Для локального деплоя подготовлен отдельный `docker-compose.local.yml` под Portainer.
- Блок настройки параллельной загрузки сделан компактнее по высоте.

### Исправлено
- Убрано краткое появление окна логина при обновлении страницы у уже авторизованных пользователей.
- Исправлены проблемы восстановления состояния плеера после перезапуска и переключения треков.
- Исправлены ошибки со старыми `stream` и `cover` ссылками после пересоздания контейнеров.
- Исправлены проблемы с отображением обложек, центрированием элементов плеера и поведением прогресс-бара.
- Сглажена анимация прогресс-бара и индикатора текущего времени в плеере во время воспроизведения и seek.
- Исправлены баги карточек плейлистов, исполнителей и длинных имён.
- Исправлены ошибки i18n для новых экранов и сценариев.
- Исправлены проблемы с определением основного исполнителя у multi-artist тегов.
- Исправлены `500` на генерации обложек после полной очистки библиотеки.
- Исправлено определение длительности новых треков при загрузке, если теги не содержат длину.

### Инфраструктура
- Версия приложения обновлена до `1.0.1`.
- Подготовлена публичная документация проекта и README для релиза.
- Docker-конфигурация приведена к режиму с отдельными Docker volumes.
